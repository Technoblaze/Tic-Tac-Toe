Tic-Tac-Toe

Tic Tac Toe. This is a game for two people. Each person will be represented with X’s and O’s. You have a 3 by 3 grid in which two players place x or o. Three characters in a row wins! Rules of the game. Each person will take a turn to mark any of a spot that is open on the bord. To win the game, a player needs to occupy 3 consecutive positions in any direction on the board.
